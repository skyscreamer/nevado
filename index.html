Introduction

Nevado JMS is a JMS driver for Amazon SQS.

  • Leverage cloud services in Amazon using standard Spring/J2EE metaphors
  • Avoid tight coupling to cloud-specific API's inside your code
  • Rapidly prototype enterprise clients in the cloud. Then deploy them anywhere.


The current version of Nevado is 1.3.0.

Nevado JMS

Our first connector supports 83% of the JMS 1.1 specification using Amazon's SQS and SNS services:

  • Written with a test-driven approach (over 300+ integration and unit tests) to ensure close adherence to JMS 1.1 specification
  • Integrates cleanly with Spring's message-driven pojos
  • Easily monitor your queues and topics through the Amazon console
